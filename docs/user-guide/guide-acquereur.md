# Guide Acquéreur — Espace Client HLM

## Bienvenue dans votre espace acquéreur

Cet espace personnel vous permet de suivre l'avancement de votre acquisition immobilière,  
consulter votre échéancier de paiement et accéder à vos documents.

---

## 1. Accéder à votre espace

### Recevoir votre lien d'accès
Votre conseiller vous envoie un **lien magique** par email dès l'ouverture de votre dossier.  
Ce lien est **unique** et valable **48 heures**.

### Ouvrir le lien
1. Ouvrez l'email intitulé *"Votre espace acquéreur HLM"*.
2. Cliquez sur le bouton **Accéder à mon espace**.
3. Vous êtes connecté automatiquement — aucun mot de passe requis.

> **Le lien a expiré ?**  
> Contactez votre conseiller pour qu'il vous envoie un nouveau lien.

### Demander un nouveau lien vous-même
1. Rendez-vous sur [votre portail](https://yem-hlm.youssouf-mehdi.workers.dev/portal/login).
2. Saisissez le code de votre société (communiqué par votre conseiller) et votre adresse email.
3. Cliquez **Envoyer le lien magique** — vérifiez vos spam si vous ne le recevez pas dans 5 minutes.

---

## 2. Tableau de bord — Mon acquisition

Une fois connecté, vous accédez à la page **Mon Acquisition** :

### Avancement du dossier
Une barre de progression visuelle montre l'état de votre vente :

| Étape | Signification |
|---|---|
| **Compromis** | Compromis de vente signé |
| **Financement** | Dossier bancaire en cours d'instruction |
| **Acte notarié** | Acte de vente signé chez le notaire |
| **Livré** | Bien remis, acquisition finalisée |

L'étape en cours est mise en évidence. Les étapes passées sont cochées.

### Informations sur votre bien
- Prix de vente convenu
- Date de compromis
- Date de l'acte notarié (une fois signé)
- Date de livraison prévue

---

## 3. Échéancier de paiement

La section **Échéancier** détaille vos versements :

| Colonne | Description |
|---|---|
| Libellé | Nom du versement (ex. Acompte 10%, Solde) |
| Montant | Montant en MAD |
| Date d'échéance | Date limite de paiement |
| Statut | En attente / Payée / En retard |

**Récapitulatif en bas de page :**
- Total payé à ce jour
- Reste à payer

> **Une échéance est marquée "En retard" ?**  
> Contactez votre conseiller dès que possible.

---

## 4. Documents

La section **Documents** rassemble vos fichiers :
- Compromis de vente
- Conditions de financement
- Acte notarié
- Tout autre document partagé par votre conseiller

Cliquez sur un document pour le **télécharger**.

---

## 5. Contact — Votre conseiller

La section **Contact** affiche les coordonnées de votre agent commercial :
- Nom et prénom
- Email
- Téléphone (si renseigné)

Vous pouvez lui envoyer un message directement depuis cette page.

---

## 6. Se déconnecter

Cliquez sur **Déconnexion** (en haut à droite ou dans le menu de navigation).  
Votre session est immédiatement fermée de façon sécurisée.

---

## Questions fréquentes

**Je ne reçois pas le lien magique.**  
→ Vérifiez vos dossiers Spam/Indésirables. Si le problème persiste, appelez votre conseiller.

**Le lien ne fonctionne plus.**  
→ Les liens expirent après 48 heures. Demandez un nouveau lien à votre conseiller ou via la page de connexion.

**Je vois "Aucune vente enregistrée pour votre dossier".**  
→ Votre dossier n'a pas encore été enregistré dans le système. Contactez votre conseiller.

**Mes informations sont-elles sécurisées ?**  
→ Oui. L'accès est protégé par un jeton unique à usage unique. Vos données sont chiffrées et accessibles uniquement par vous et votre conseiller.
