# Administration Guide

This guide is for admins, managers with elevated responsibilities, and `SUPER_ADMIN`.

## 1. User And Invitation Management

Company administrators can:

- list members
- invite new users
- resend invitations
- update roles where allowed
- remove or deactivate members
- unblock users

Important rule:

- only `SUPER_ADMIN` can assign the `ADMIN` role

## 2. Template Management

Template management allows admins to control generated content and previews.

Use it to:

- review template sources
- edit the active template
- preview the rendered result
- reset or clean up templates when needed

## 3. Superadmin Societe Management

`SUPER_ADMIN` can:

- create a societe
- update branding and legal identity
- review stats and compliance information
- inspect members
- suspend or reactivate the societe

## 4. Impersonation

Impersonation is a support tool.

Use it when:

- you must reproduce a role-specific issue
- you need to verify what a user can actually see

Best practice:

- use it deliberately
- end impersonation as soon as the support action is complete

## 5. Governance Advice

- review roles before inviting users
- keep template edits controlled and traceable
- separate platform support actions from day-to-day CRM operations
- use audit and privacy workflows when handling sensitive requests
