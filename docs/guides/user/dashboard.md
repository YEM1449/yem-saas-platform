# Dashboard Guide

The dashboard area helps staff turn operational data into action.

## 1. Dashboard Areas

### Home and commercial views

Use these views to monitor:

- sales activity
- pipeline volume
- conversion indicators
- alerting and trend slices

### Cash and receivables views

Use these views to monitor:

- due and overdue payment items
- collection pressure
- cash expectations
- payment follow-up priorities

### Commission view

Use this area to review:

- commission totals
- project-specific rule effects
- agent-scoped performance when available to your role

## 2. How To Use The Dashboard Well

- start with the summary cards
- identify anomalies or overdue situations
- drill into the detailed view that matches the issue
- move from KPI observation to action in contacts, ventes, or tasks

## 3. Typical Questions The Dashboard Answers

- are we closing enough deals this period?
- which agents or projects need attention?
- which payments are overdue?
- are pipeline and cash views telling the same story?

## 4. Role Expectations

- admins and managers generally have the broadest analytics view
- agents see relevant scoped information for their work
- dashboard numbers are tied to the active societe

## 5. Best Practices

- use the dashboard at the start of the day
- cross-check dashboards with the underlying vente or payment record before making decisions
- create tasks or follow-up actions directly after spotting a problem
